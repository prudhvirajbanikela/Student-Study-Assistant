# 🎓 Student Study Assistant

An AI-powered educational platform that combines Conversational AI, Document Intelligence, OCR, and Data Analytics to create a smarter and more interactive learning experience for students.

## 🚀 Overview

Student Study Assistant is designed to help students analyze academic content, explore datasets, summarize study materials, and interact with uploaded documents through a ChatGPT-style conversational interface.

The platform leverages Large Language Models (LLMs), Optical Character Recognition (OCR), and Data Analytics to transform static study resources into intelligent, interactive learning tools.
